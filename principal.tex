\begin{abstract}
\noindent

O presente trabalho tem como objetivo a criação de um protótipo de comandas de casas de Happy Hour que utiliza as novas tecnologias Web por meio da abordagem de desenvolvimento \textit{Progressive Web Apps} (PWA). O aplicativo proposto é suportado por todas as plataformas que possuem um browser com suporte as tecnologias da Web moderna, resultando em uma nova percepção para o desenvolvimento mobile. A PWA traz a proposta de resolver problemas das abordagens atuais proporcionando a mesma experiência imersiva e intuitiva para os dispositivos mobile. Através do desenvolvimento e testes foi possível elencar os pontos positivos e negativos ao se adotar essa abordagem na concepção e criação de sistemas mobile.

\\\textbf{\keyword{Palavras-chave: }}\textit{Progressive Web Apps}; PWA; Dispositivos Móveis; Desenvolvimento de Software; Desenvolvimento Mobile.
\end{abstract}
